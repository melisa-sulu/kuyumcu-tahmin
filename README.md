# 💎 Kuyumcu Mücevher Değeri Tahmin Sistemi

> AI destekli profesyonel kuyumcu yönetim platformu

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Flask](https://img.shields.io/badge/Flask-2.3+-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 📸 Proje Hakkında

Yapay zeka destekli kuyumcu yönetim sistemi. Mücevher fotoğrafından otomatik fiyat tahmin et, müşteri yönetimi yap, stok takibi et ve piyasa analizi incele.

**Staj Projesi:** 2 aylık kuyumculuk stajı deneyimi ile geliştirildi.

---

## ✨ Özellikler

- ✅ **Fiyat Tahmin** - Mücevher fotoğrafından otomatik fiyat tahmini
- ✅ **Müşteri CRM** - Müşteri bilgileri ve sipariş takibi
- ✅ **Stok Yönetimi** - Mücevher envanteri ve uyarıları
- ✅ **Piyasa Analizi** - Altın/Gümüş fiyat grafiği ve trend analizi
- ✅ **Sipariş Takibi** - Sipariş durumu ve ödeme takibi
- ✅ **Responsive Design** - Mobil uyumlu arayüz

---

## 🛠️ Teknolojiler

**Backend:**
- Flask 2.3.3 (Python Web Framework)
- SQLAlchemy (ORM)
- SQLite (Database)

**Machine Learning:**
- Scikit-learn (Fiyat tahmini modeli)
- Pandas (Veri analizi)
- NumPy (Matematiksel işlemler)

**Frontend:**
- Bootstrap 5 (UI Framework)
- HTML/CSS/JavaScript
- Font Awesome (İkonlar)

**Görüntü İşleme:**
- OpenCV (Fotoğraf analizi)

## 📦 Kurulum

### Gereksinimler
- Python 3.9+
- pip
- Mac/Linux/Windows

### Adımlar

**1. Repo'yu klonla:**
```bash
git clone https://github.com/SENIN-GITHUB-ADINA/kuyumcu-tahmin.git
cd kuyumcu-tahmin
```

**2. Virtual Environment oluştur:**
```bash
python3 -m venv venv
```

**3. Aktif et:**
```bash
# macOS/Linux:
source venv/bin/activate

# Windows:
venv\Scripts\activate
```

**4. Kütüphaneleri yükle:**
```bash
pip install -r requirements.txt
```

**5. Uygulamayı çalıştır:**
```bash
python app.py
```

**6. Tarayıcıda aç:**

## 📊 Kullanım

### Ana Sayfa
- Hızlı istatistikler (Müşteri, Ürün, Sipariş sayısı)
- Hızlı erişim butonları

### Fiyat Tahmin
1. Mücevher fotoğrafını yükle
2. Kategori ve malzeme seç
3. Tahmini ağırlık gir
4. "Fiyat Tahmin Et" tuşuna tıkla
5. Tahmini fiyat ve detaylar gösterilir

### Müşteri Yönetimi
- Yeni müşteri ekle
- Müşteri bilgilerini görüntüle
- Satış geçmişini takip et

### Stok Yönetimi
- Ürün ekle/sil/düzenle
- Toplam stok değerini görüntüle
- Stok uyarıları (kırmızı: <5, sarı: <10)

### Analiz Paneli
- Piyasa fiyatları (Altın, Gümüş)
- Aylık satış istatistikleri
- Kar marjı analizi
